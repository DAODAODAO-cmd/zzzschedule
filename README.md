# 🛒 谷子排单簿

欢迎使用排单簿 ✨

把评论、排单、调价、推车图和收款统计放在一起，让排单变得更轻松一点。

---

## 🌟 可以做什么

- 💬 批量粘贴评论自动排单
- ➕ 新评论直接追加，不会覆盖原来的数据
- ✏️ 表格可以直接修改、拖动调整
- 💰 自动计算调价、均价和收款金额
- 🖼️ 一键生成排单图、推车图
- 📱 手机、电脑都可以使用
- ☁️ 数据自动同步，不用重复录入

---

## 📖 使用方法

### 第一次使用

① 新建一个排单

② 输入（或选择）同步码

③ 开始排单

---

### 日常排单

1️⃣ 添加角色

2️⃣ 粘贴评论

3️⃣ 检查排单结果

4️⃣ 根据需要调整

5️⃣ 导出推车图或排单图

---

## ☁️ 同步

大家使用同一个同步码即可查看和编辑同一份排单。

修改后会自动保存，其他设备也可以继续接着排。

---

## 💛 小提醒

- 建议偶尔导出备份。
- 推车图导出前记得检查价格和数量。
- 同步码请不要随意分享给其他人。
